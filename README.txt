video face blur demo (webcam realtime)
using res10_300x300_ssd_iter_140000.cafeemodel
opencv
thanks to pyimagesearch (adrian@pyimagesearch.com for blur functions)
=====================================================================


opencv
We use opencv to read and write webcam frames, to get images from webcam and detect 'face' using classifiers

1- install python3 (download pip distribution)

pip3 install tensorflow --upgrade

2- install pip modules
pip install -r dependencies.txt

3- execute main.py

python main.py

	key control:(mouse focus must be in webcam window)
	ESC -> Exit
